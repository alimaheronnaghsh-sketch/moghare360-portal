# PHASE 4 — Supplier Flow

Page: `public_html/erp-supplier-board.php`

## Features

- Lists suppliers from `erp_suppliers`
- Self-POST supplier create (CSRF `inventory_supplier_create`)
- Lists purchase requests with inline status update forms
- No financial payment processing

## Supplier Fields

`supplier_name`, `supplier_code`, `supplier_type`, `contact_name`, `mobile`, `phone`, `address_text`, `notes`

## Seed

`INTERNAL-MANUAL` — تامین‌کننده دستی داخلی (from SQL)
