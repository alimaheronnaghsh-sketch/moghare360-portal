# JobCard UX Scope

## In Scope
- JobCard workbench (list + KPI)
- JobCard detail UX (summary, binding, related data)
- Combined timeline UX from history tables
- Create UX guide (reception flow, mock form, link to M17)
- JobCard UX CSS
- Read-only SELECT from existing tables

## Out of Scope
- SQL schema / migration
- DB write on UX pages
- Auth / login / permission model change
- Direct role assignment
- Finance / inventory / QC / delivery write from UX pages
- Legacy portal change
- Production deploy

## Final Scope Decision
Mission 33 = JobCard UX wrapper layer with read-only data display.
