# MOGHARE360 Persian Language Guide

## زبان اصلی

زبان اصلی محصول **فارسی مدیریتی** است. UI باید برای مدیر تعمیرگاه، سرمایه‌گذار و شریک کاری قابل فهم باشد.

## اصطلاحات فنی

اصطلاحات فنی کنترل‌شده مجازند، اما کنار آن‌ها توضیح فارسی باید وجود داشته باشد. حذف کور انگلیسی ممنوع است.

## قانون انگلیسی

انگلیسی فقط وقتی مجاز است که اصطلاح فنی شناخته‌شده باشد و کنار آن توضیح فارسی وجود داشته باشد.

مثال صحیح: `KPI (شاخص کلیدی عملکرد)`  
مثال نادرست: حذف KPI و جایگزینی بدون context
