# PHASE 15 SCOPE

## Allowed

- Demo Package ZIP
- Local Release Package ZIP
- Release dashboard and download pages
- Manifest and release notes docs
- Packaging PowerShell scripts (no auto deploy)

## Forbidden

- Production installer
- Auto deploy overwriting real config
- Credentials in package
- Private config in package
- DB backup with sensitive data
- Real customer data
- SaaS / portal / accounting activation

See PHASE_15_01_SCOPE integration list in repo pages.
