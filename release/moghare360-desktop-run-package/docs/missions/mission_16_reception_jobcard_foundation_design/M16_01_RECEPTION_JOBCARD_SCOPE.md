# Reception / JobCard Scope

## Purpose
This document defines the Reception and JobCard foundation scope for MOGHARE360 ERP Soft Run.

## Included Scope
Mission 16 designs:
- Reception flow
- JobCard foundation
- JobCard status model
- Customer / Vehicle link rules
- Service operation boundary
- Permission and workflow rules
- Future SQL implementation plan
- Future UI plan
- Future testing plan

## Excluded Scope
Mission 16 does not include:
- SQL execution
- PHP implementation
- JobCard create form
- Service operation write
- Parts / inventory write
- Finance write
- Delivery / settlement
- Customer Portal
- Customer login
- Legacy migration
- Production deploy

## Soft Run Goal
Soft Run must support controlled internal creation of a JobCard for an existing Customer and Vehicle.

## Final Scope Decision
Reception and JobCard foundation is locked as an internal ERP workflow foundation, not a public portal or finance module.
