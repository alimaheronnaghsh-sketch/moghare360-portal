# PHASE 7 — Scope

## In Scope
- HR foundation tables (`erp_hr_*`)
- Employee profile and controlled create
- Employment contract records
- Attendance records with hour calculation
- Payroll **preview** (internal, non-official)
- Training and disciplinary records
- HR management dashboard
- Internal staff portal preview (via existing ERP auth)

## Out of Scope (Forbidden)
- Real payroll payment
- Insurance / tax integration
- Ministry of Labor official documents
- Legal payroll slip
- Real messaging
- Biometric / attendance device integration
- Self-service staff login portal
- Production login/auth/permission changes

## Employee Lifecycle Blueprint
ورود → قرارداد کاری → حضور و غیاب → کارکرد → آموزش → ترفیع/تنبیه → فیش حقوقی preview → پروفایل پرسنلی
