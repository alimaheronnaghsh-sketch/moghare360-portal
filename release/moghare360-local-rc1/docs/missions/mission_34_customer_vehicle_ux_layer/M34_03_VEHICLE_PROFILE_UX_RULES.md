# Vehicle Profile UX Rules

## Page
`erp-vehicle-detail-ux.php`

## ID Resolution
- `?vehicle_id=` from querystring
- Default: first vehicle in erp_vehicles if missing/invalid

## Sections
1. Vehicle profile card (plate, VIN, brand, model, mileage)
2. Customer binding cards (relations)
3. Parts / payment / QC count summaries
4. JobCard history for vehicle
5. Service operations linked via jobcards
6. Action panel links

## Final Vehicle Decision
Vehicle-centric view with owner binding and service history access.
