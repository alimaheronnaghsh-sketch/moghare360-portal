# PHASE 12.5 SCOPE

## Goal

Make MOGHARE360 more Persian, branded, presentable, and copyright-controlled.

## Allowed

- Persian localization of main pages (audit/registry)
- Official product dictionary
- Moghareh Motors branding
- Asset registry
- Copyright policy
- Product language guide
- Demo package page
- Safe links on non-sensitive product pages

## Forbidden

- Production login rewrite
- Auth architecture rewrite
- Destructive DB migration
- Legacy portal change
- Customer public portal activation
- Real SaaS activation
- Payment gateway / billing
- Official accounting / tax invoice
- External SMS/WhatsApp/Email/BI
- Dependency installation
- Third-party copyrighted assets without license
- Car brand logos
- Unlicensed font files

## No SQL

Static/read-only layer preferred.
