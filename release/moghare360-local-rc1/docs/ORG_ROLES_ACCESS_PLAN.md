# MOGHARE360 Organization Roles And Access Plan

## Goal

Every user must have a role, department, position, permission level, and workflow responsibility.

## Departments

1. Executive Management
2. Operations
3. Reception
4. CRM
5. Mechanical
6. Electrical and Options
7. Suspension and Undercarriage
8. Technical Management
9. Inventory
10. Purchase
11. Finance
12. HR
13. Marketing and Sales
14. System Administration

## Core Rule

No user should access all pages unless they are owner, admin, or approved manager.

## Access Levels

1. Owner
2. General Manager
3. Operations Manager
4. Department Manager
5. Staff
6. Read Only
7. Customer
