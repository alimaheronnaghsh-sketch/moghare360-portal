# MOGHARE360 V1 — Run First (فارسی)

## ۱. Desktop (لپ‌تاپ / سیستم مقصد)
1. Extract `moghare360-desktop-run-package.zip`
2. `CHECK_REQUIREMENTS.ps1`
3. `INSTALL_LOCAL_COPY.ps1` در صورت نیاز
4. تنظیم `private/erp-config.php` روی سیستم مقصد
5. `START_MOGHARE360.bat`

URL: `http://localhost:8080/moghare360/`

## ۲. Mirror (moghareh360.ir)
1. Extract `moghare360-mirror-site-package.zip`
2. آپلود `public_html/`
3. تنظیم `mirror-config.php` از example
4. تست `mirror-health.php`

## ۳. قوانین
- داده فقط روی Master Laptop
- هاست بدون دیتابیس
- بدون Cloud storage
